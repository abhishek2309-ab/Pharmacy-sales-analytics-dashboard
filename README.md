# 💊 Pharmacy Sales Analytics Dashboard

This project is a **Power BI Dashboard** built to analyze pharmacy sales data and generate actionable business insights such as revenue trends, profit distribution, and category performance.

## 🚀 Key Metrics

- 💰 Total Revenue: **16.29M**
- 📈 Total Profit: **3.67M**
- 📦 Quantity Sold: **21K**
- 🧾 Average Order Value: **8.14K**

---

## 📌 Features

- 📅 Date range filtering
- 🏢 Branch-wise analysis (Pune, Mumbai, Hyderabad, Delhi, Bangalore)
- 💊 Category-wise performance tracking
- 📈 Monthly revenue trends
- 📊 Interactive charts & KPI cards

---

## 📊 Dashboard Insights

- Hyderabad and Mumbai are top revenue-generating branches
- Antibiotics & Painkillers dominate category sales
- Revenue shows slight declining monthly trend
- Strong profit margin of ~18%
